# CET-6 词汇打卡训练器

## Overview

本技能用于大学英语六级（CET-6）词汇的日常打卡训练。采用「选模式 → 出题 → 答题 → 批改 → 错题收录 → 间隔复习」的闭环学习模式，确保答错的单词在后续练习中反复出现，直到掌握为止。

**开始练习前先显示模式选择菜单**，用户可二选一：
- **模式 A：中译英（拼写练习）** — 给中文释义，写英文单词，强化拼写记忆
- **模式 B：英译中（认读练习）** — 给英文单词，写中文意思，强化阅读识别

默认每次练习 20 题。错题自动存入错题本，下次出题时优先抽取。

---

## 工作流程

### 第一步：显示模式选择菜单

用户开始练习时，先展示模式选择菜单，让用户二选一：

```
📚 CET-6 词汇打卡 — 请选择练习模式：

  1️⃣  中译英（拼写练习）
      给中文释义 → 写英文单词
      适合：强化拼写、备考写作翻译

  2️⃣  英译中（认读练习）
      给英文单词 → 写中文意思
      适合：快速认读、备考阅读

回复数字 1 或 2 开始～
```

用户选择后，按对应模式生成题目。

### 第二步：生成练习题

每次出题时按以下优先级选取单词：

1. **错题优先**：从错题本中抽取 30%-50% 的单词（优先选取错误次数最多的）
2. **新词补充**：从词汇库 `references/word-bank.md` 中选取剩余的新词
3. **模式统一**：全部为同一种题型（全中译英 或 全英译中）
4. **难度均衡**：覆盖动词、形容词、名词等不同词性

**出题格式要求：**
- 编号清晰（1-20）
- 中译英题：给出中文释义 + 词性提示
- 英译中题：给出英文单词 + 词性
- 不要在题目中泄露答案
- 标题标注模式：「中译英拼写练习」或「英译中认读练习」

### 第三步：用户作答

用户按编号提交答案，格式示例：
```
1. xxx 2. xxx 3. xxx ...
```

### 第四步：批改与讲解

逐题批改，分为三档：

| 等级 | 判定标准 | 是否记入错题 |
|------|----------|-------------|
| ✅ 完全正确 | 拼写正确、意思准确、词性匹配 | 否 |
| ⚠️ 不够精准 | 意思接近但非最佳答案、词性偏差、简单词替代六级高频词 | 是（标记为 close_but_imprecise） |
| ❌ 错误 | 拼写错误、意思完全错误、词性完全不对 | 是（标记为 spelling 或 meaning） |

**批改输出结构：**
1. 成绩总览（正确率统计）
2. 错题详解（每个错词包含：正确答案、辨析、记忆技巧、例句）
3. 不够精准的词（给出更优答案和区别）
4. 今日重点词汇表（建议抄写复习）
5. 鼓励性结尾

### 第五步：记录错题

批改完成后，将错误和不够精准的单词写入错题本。使用 `scripts/wrong_words.py` 脚本或直接维护 JSON 记录。

每条错题记录：
- 单词、中文释义
- 错误类型（spelling / meaning / part_of_speech / close_but_imprecise）
- 错误次数
- 首次/最近一次错误时间
- 用户的错误答案（便于分析）

### 第六步：错题复习机制

**核心规则：答错的词在后续练习中重复出现，直到连续答对 2-3 次才标记为已掌握。**

- 错误次数越多的词，出现频率越高
- 每次新练习中，错题占比建议 30%-50%
- 连续答对 2 次后，降低出现频率
- 连续答对 3 次后，标记为 mastered（已掌握），移出常规复习池
- 拼写类错题优先安排在「中译英模式」复习，认读类错题优先安排在「英译中模式」复习

## 出题规范

### 模式 A：中译英（拼写练习）
- 题目格式：`序号. 中文释义（词性）`
- 重点考察：拼写易错词、形近词、六级高频动词/形容词
- 示例：`1. 减轻，缓解（v.）`
- 适合场景：写作、翻译备考，强化手写记忆

### 模式 B：英译中（认读练习）
- 题目格式：`序号. word (词性)`
- 重点考察：熟词僻义、易混词、抽象名词
- 示例：`1. alleviate (v.)`
- 适合场景：阅读备考，快速扩大认读量

### 词汇选取原则
1. 优先从 `references/word-bank.md` 中选取，确保六级难度
2. 覆盖不同词性（动词、形容词、名词为主）
3. 避免同一次练习中出现近义词对（除非是辨析题）
4. 错题复习时，按错误类型匹配模式：拼写错误 → 中译英模式，词义错误 → 英译中模式

---

## 批改标准详解

### ✅ 算对的情况
- 拼写完全正确，意思吻合
- 近义词且属于同等难度级别（如 "huge" 和 "enormous" 都表示巨大，但六级应考 enormous）

### ⚠️ 不够精准（需要指出更优答案）
- 用简单词替代了六级高频词：如 start → embark, general → prevalent
- 词性略有偏差但意思对：如 suppression（名）→ suppress（动）
- 意思接近但适用场景不同：如 ambiguous vs vague

### ❌ 算错的情况
- 拼写错误（字母顺序错、漏字母、多字母）
- 意思完全不相关
- 词性完全不匹配（题目要动词答了名词且意思也不对）

## 错题管理脚本

使用 `scripts/wrong_words.py` 管理错题本，数据存储在 `~/.cet6_wrong_words.json`。

### 常用命令

```bash
# 添加错题
python scripts/wrong_words.py add "dilemma" "困境，进退两难" "spelling" "delimma"

# 查看所有错题（按错误次数排序）
python scripts/wrong_words.py list

# 获取下一次要复习的单词（默认10个）
python scripts/wrong_words.py review --count 8

# 标记单词为已掌握
python scripts/wrong_words.py mastered "dilemma"

# 查看统计数据
python scripts/wrong_words.py stats
```

### 也可以直接操作 JSON
如果脚本调用不便，可直接读取/修改 `~/.cet6_wrong_words.json` 文件，结构如下：

```json
{
  "dilemma": {
    "word": "dilemma",
    "meaning": "困境，进退两难",
    "error_type": "spelling",
    "error_count": 2,
    "first_error": "2025-06-30T08:00:00",
    "last_error": "2025-07-01T08:00:00",
    "wrong_answers": ["delimma", "dilema"],
    "mastered": false
  }
}
```

## 词汇库使用

完整词汇库见 `references/word-bank.md`，按词性分类收录六级高频词：
- 动词（20+）：alleviate, disseminate, embark, suppress, refrain...
- 形容词（20+）：ambiguous, prevalent, prudent, skeptical, tedious...
- 名词（20+）：dilemma, scrutiny, prestige, omen, consensus...
- 短语搭配（12 组）：account for, attribute to, cope with...
- 出题优先级建议

出题时优先从词汇库选取，确保难度稳定。词汇库可根据用户水平动态扩充。

## 定时打卡场景

当配合定时任务（如每日早 8 点）使用时：
1. 定时任务触发后，先显示模式选择菜单（1=中译英，2=英译中）
2. 用户选择模式后生成对应题目（20 题）
3. 题目中包含 30%-50% 的复习错题
4. 用户回复答案后触发批改 + 错题更新

如果定时任务需要直接出题不选模式（如用户已固定偏好），可默认使用「中译英模式」，因为拼写练习对记忆强化效果更好。

## 学习建议（可在批改后附带）

1. **错题一定要手写一遍**：拼写错误光看不写记不住
2. **结合例句记单词**：孤立记词容易忘，放在句子里更牢固
3. **艾宾浩斯复习节奏**：当天、第 2 天、第 7 天、第 14 天复习效果最佳
4. **认读和拼写并重**：阅读靠认读，写作翻译靠拼写，两者都要练
